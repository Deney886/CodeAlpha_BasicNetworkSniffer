# Basic Network Sniffer

## Objective
Capture and analyze network packets using Python and Scapy.

## Tools Used
- Python
- Scapy
- Npcap

## Features
- Captures network packets
- Displays Source IP
- Displays Destination IP
- Displays Protocol
- Displays Payload

## How to Run
pip install scapy

python network_sniffer.py
